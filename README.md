# FTP
L’objectif du projet est de construire un serveur de fichiers inspiré par le protocole FTP

# Compilation : 
make clean -> make 

# Test : 

Serveur (Terminal 1) : ./ftpserveri

Client (Terminal 2) : 
    ./ftpclient 127.0.0.1
    get lorem.txt 


# Remarque : 
127.0.0.1, correspond au localhost
